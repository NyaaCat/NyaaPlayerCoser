# NyaaPlayerCoser
NyaaPlayerCoser (NPC) plugin.

# Quick command guide

    /npc spawn VILLAGER TRADER_UNLIMITED TestNpc
    
    /npc edit help
    /npc edit 0 trade:+
    /npc edit 0 trade:+<existing_trade_id>
    /npc edit 0 trade:-<trade_id>
    
    /npc inspect nearby
    /npc inspect nearby 5
    /npc inspect npc 0
    /npc inspect trade 0
